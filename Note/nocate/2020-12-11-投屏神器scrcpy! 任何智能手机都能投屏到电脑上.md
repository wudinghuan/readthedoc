# 连接手机

用usb线连接手机到电脑, 打开cmd, 输入以下命令

```
adb devices
```

看到有自己的手机设备即连接成功

![投屏神器scrcpy! 任何智能手机都能投屏到电脑上](https://p3-tt.byteimg.com/origin/pgc-image/16083dea24c3467cb2d97081c02f22f1?from=pc)



# 开始投屏

在cmd中输入

```
scrcpy
```

回车即可
